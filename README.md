# 📦 Controle de Estoque

## 💻 Sobre o Projeto

Este projeto é um sistema simples de controle de estoque desenvolvido em Python.
Ele permite cadastrar produtos, armazenar informações como preço e estoque mínimo, além de atualizar a quantidade disponível.

---

## 🚀 Funcionalidades

* ➕ Cadastro de novos produtos
* 🔄 Atualização da quantidade em estoque
* 💰 Registro de preço do produto
* ⚠️ Definição de estoque mínimo
* 📋 Listagem final dos produtos cadastrados

---

## 🛠️ Tecnologias Utilizadas

* Python 3.8+
* Biblioteca padrão (sem dependências externas)

---

## ▶️ Como Executar o Projeto

1. Execute o arquivo principal:

```bash
python main.py
```

2. Informe:

* Nome do produto
* Quantidade
* (Caso novo) Preço e estoque mínimo

---

## 📌 Observações

* Os dados são armazenados apenas em memória (não persistem após encerrar o programa)
* O sistema funciona via terminal
