# Atualização de Estoque **:package:**

## **:computer:** Sobre o Projeto

Projeto de controle de estoque cujo objetivo é permitir que o usuário selecione um produto e **adicione** uma quantidade ao seu estoque atual. Não faz remoção, nem vendas, apenas incremento de quantidade.


## 🛠️ Tecnologias Utilizadas

-   Python 3.8+
-   Biblioteca padrão (nenhuma dependência externa)


##  **:sparkles:** Funcionalidades

- Adicionar quantidade a um produto existente.
- Listar produtos cadastrados.
